Sandbox

Free-input component: a labeled text input (or textarea) plus an output region. The lesson author supplies the evaluation logic; the component handles debounced wiring and error surfacing.

Regex tester

Type a JavaScript regex pattern. The sandbox calls the author-provided vtDemoTestRegex function and renders the result live.

Temperature converter

Enter a Celsius value; the sandbox converts it to Fahrenheit using the vtDemoTempConvert hook.

Multi-line textarea

Enter one word per line; the sandbox counts distinct words via vtDemoWordCount.

Error surfacing

This sandbox uses vtDemoMaybeThrow, which throws when the input is not a number. Try typing a letter.
