Teacher Box

A prompted self-explanation block with question chips that surface common learner questions.

Example

Before moving on — can you explain it?

Try to answer one of these in your own words. Explaining a concept is the fastest way to find the gaps in your understanding.

What problem does this solve? When would you use this over the alternative? What breaks if you skip this step?
