Callout

Five semantic tones for surfacing warnings, information, insights, successes, and risks inline with prose.

Warning (default) — Double-check your configuration before deploying. Misconfigured environment variables can cause silent failures in production.
Info — This API is available in all regions. Rate limits apply per project, not per user account.
Insight — Batching multiple writes into a single transaction can reduce round-trips by up to 80% on high-latency connections.
Success — Your migration completed without errors. All 42,000 rows were transferred and verified.
Risk — Deleting this record will cascade to 17 dependent rows. This action cannot be undone.
