diagram · wide SVG · FORGOTTEN → #9: max-width:100% shrinks it, overflow-x:auto can't fire

Real inline SVG (intrinsic 1200px) in the measure column. svg{max-width:100%} crushes it to fit — no scrollbar, because nothing overflows. Simple diagrams survive the shrink; dense ones go to mush. Table + code (above) instead keep size and scroll, because overflow-x:auto isn't defeated for them. So a forgotten .vt-wide on a wide diagram is opt-in's one real-but-mild wound.
