Figure / photo block

Framed images with styled captions, and before/after photo pairs.

Pure CSS · responsive · light + dark

1.vt-figure — single framed image

Wrap a <figure> in .vt-figure to get a themed border/radius matching other vt-* blocks, a centered responsive image, and a caption visually distinct from body prose.

2Broken / missing image — placeholder state

When the image is unavailable the alt text renders inside the same themed frame — a soft background with muted text that works in both light and dark mode.

3.vt-figure-pair — before / after side-by-side

Two .vt-figure elements inside .vt-figure-pair sit side-by-side on wide viewports and stack vertically below 560 px. Add .before / .after on each figure for red/green label cues.

4.vt-figure-pair — both images missing

The placeholder state in a pair — same themed background in both columns, readable in light and dark mode.
