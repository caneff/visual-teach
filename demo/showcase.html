fibonacci.py 
def fibonacci(n):
    """Return the nth Fibonacci number."""
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b

# Print first 10 Fibonacci numbers
for i in range(10):
    print(f"fib({i}) = {fibonacci(i)}")

Display math block — quadratic formula

x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}

The quadratic formula gives the roots of any polynomial \(ax^2 + bx + c = 0\) where \(a \neq 0\).

Euler's identity

e^{i\pi} + 1 = 0

Often called the most beautiful equation in mathematics, it connects five fundamental constants: \(e\), \(i\), \(\pi\), \(1\), and \(0\).

Normal distribution PDF

f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)

The probability density function of a normal distribution with mean \(\mu\) and standard deviation \(\sigma\).

Inline math in prose

The Pythagorean theorem states that in a right triangle, \(a^2 + b^2 = c^2\), where \(c\) is the hypotenuse. For example, a 3-4-5 triangle satisfies \(9 + 16 = 25\).
