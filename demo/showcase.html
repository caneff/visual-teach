9Mermaid (opt-in computed graphs — CDN loads only on this page)

When to use: mermaid is for graphs that must be auto-laid-out (sequence, state machine, flowchart, class, ER). The brand-blue theme is tuned for these box-and-arrow types; multi-series charts (pie, xy) come out mono-blue and should use a native theme instead. For anything you can hand-compose, use the CSS diagram vocabulary — it prints cleanly with zero JS overhead.
