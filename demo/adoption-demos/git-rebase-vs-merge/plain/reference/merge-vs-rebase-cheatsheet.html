The one decision

Ask this every time

"Have these commits been shared with anyone yet?"
No → rebase freely (private, tidy it up).  Yes → merge (don't rewrite what others have).

Conflict-during-rebase routine

git rebase main
  └─ CONFLICT?
       1. edit the file, resolve <<</===/>>> markers
       2. git add <file>
       3. git rebase --continue      (repeat per conflicting commit)
     ...or at any point:
          git rebase --abort         (restores pre-rebase state)

The Golden Rule

Never break this

Never rebase commits that have been pushed and shared. Rewrite private commits all you like; never rewrite ones others may have built on. [Pro Git §3.6]

Glossary

Commit
A snapshot of your project plus a pointer to its parent commit(s). Identified by a hash computed from its content and its parent.
Commit hash (SHA)
The unique ID of a commit. Because it depends on the parent, replaying a commit onto a new base yields a new hash — why rebase makes C′.
Base
The commit a branch starts from. Rebase changes the base, then replays your work on top of the new one.
Merge commit
A commit with two parents, created by git merge to join two lines of history. The "knot" that preserves the branching shape.
Fast-forward
When the target branch hasn't diverged, git just slides the branch pointer forward — no merge commit needed.
Replay
What rebase does: it re-applies each of your commits, in order, on top of the new base, creating new commits as it goes.
Linear history
A history with no forks — one straight line of commits. The product of rebasing. Easy to read; hides that parallel work ever happened.
Reflog
Git's private log of every position HEAD has held, kept for weeks. The safety net that makes almost any rebase recoverable. [docs]
Golden Rule (of rebasing)
Never rebase commits that have been pushed and shared. The single rule that keeps rebase from clobbering teammates' work.
Shared vs. private commits
A commit is shared once pushed where others may have pulled it; until then it's private. The whole merge-vs-rebase safety call.
