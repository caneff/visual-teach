The 10-second model

Make a container (display: flex). Its direct children become items flowing along the main axis (flex-direction). The cross axis is perpendicular. Distribute along main with justify-content, align across cross with align-items, size items with flex, reflow with flex-wrap.

Which property goes where

On the container

display, flex-direction, flex-wrap, justify-content, align-items, gap

On the items

flex (= flex-grow / flex-shrink / flex-basis), align-self, order

Container properties

PropertyValuesWhat it does
displayflex Turns the element into a flex container; direct children become items.
flex-direction row · column · row-reverse · column-reverse Sets the main axis. row = horizontal (default), column = vertical.
flex-wrap nowrap · wrap · wrap-reverse nowrap (default) forces one line; wrap lets overflow drop to new lines.
justify-content flex-start · flex-end · center · space-between · space-around · space-evenly Positions / distributes items along the main axis.
align-items stretch · flex-start · flex-end · center · baseline Aligns items along the cross axis. stretch is the default.
gap <length> · row-gap column-gap Fixed space between items only — never at the outer edges.

Item properties

PropertyValuesWhat it does
flex<grow> <shrink> <basis> Shorthand for the three below. Memorise the three presets in the next table.
flex-grow0 (default) · <number> Share of leftover space this item claims. Ratios, not pixels.
flex-shrink1 (default) · 0 · <number> How readily it shrinks when space is tight. 0 = never below basis.
flex-basisauto (default) · 0 · <length> Starting main-axis size before grow/shrink applies.
align-selfsame as align-items Overrides align-items for this one item on the cross axis.
order0 (default) · <integer> Reorders visually without touching the HTML. Lower renders first.

The three flex presets to memorise

ShorthandExpands toUse when
flex: 11 1 0 You want equal-width columns that fill the row.
flex: auto1 1 auto Items should grow from their natural content size.
flex: none0 0 auto A fixed item (icon, button) that must never grow or shrink.
flex: 1 1 200px(as written) Responsive cards: aim ~200px each, grow/shrink/wrap to fit.

Recipes for the four mission tasks

GoalCSS on the container
Center a box, both directions display:flex; justify-content:center; align-items:center;
Nav bar: logo left, menu right display:flex; justify-content:space-between; align-items:center;

Glossary

The shared vocabulary for the whole course. Every lesson uses these terms exactly as defined here.

Flex container
An element with display: flex. It lays out its direct children as flex items. Lesson 1.
Flex item
A direct child of a flex container. Grandchildren are not flex items — they're laid out by their own parent.
Main axis
The line items flow along, chosen by flex-direction. Horizontal for row, vertical for column.
Cross axis
The axis perpendicular to the main axis. Never set directly — it's defined by whatever the main axis is.
justify-content
Container property that positions and distributes items along the main axis. Lesson 2.
align-items
Container property that aligns items along the cross axis. Centering = both this and justify-content set to center.
gap
Fixed spacing between items, set on the container. Distinct from space-*, which distributes variable leftover space.
flex-grow / flex-shrink / flex-basis
The three components of the flex shorthand on an item: how it takes extra space, gives space back, and what size it starts from. Lesson 3.
flex-wrap
Container property deciding whether items stay on one line (nowrap, default) or reflow onto multiple lines (wrap).
