CSS Flexbox · Lesson 2

Distributing and aligning

The workhorse skill: position items along the main axis, align them on the cross axis, and finally center a box on purpose.

⏱ ~15 min Needs: Lesson 1 Beginner Lesson 2 of 3

By the end of this lesson you'll be able to:

  • Distribute items along the main axis with justify-content.
  • Align items on the cross axis with align-items — and center a box both directions.
  • Add even spacing with gap.
  • Read a target layout and pick the properties that produce it, the first time.

1justify-content — along the main axis

justify-content decides how items sit along the main axis (horizontal in a default row). The first three pack the items together; the last three spread the free space between them.

ValueGap at the two outer edgesGaps between items
space-betweennoneequal

2align-items — along the cross axis

align-items positions items on the cross axis (vertical in a default row). Its default is stretch — items grow to fill the container's height. Set it to center and the items sit in the vertical middle. This is the property that finally centers things vertically — no position, no negative margins.

The centering recipe. To center a box both directions, center it on each axis at once: display:flex; justify-content:center; align-items:center;. Main axis centers it horizontally, cross axis centers it vertically. The thing you could never reliably do is now two words.

3gap — breathing room between items

gap sets space between items — and only between them, never on the outer edges. It replaces the old trick of margins on every child (and then fighting the leftover margin on the last one). One property on the container does it for the whole row.

5Lock it in

You want a row's items centered vertically. Which property and value?

From memory — say it before you reveal:

Name the two declarations that center a single box both horizontally and vertically.
justify-content: center (main axis → horizontal) and align-items: center (cross axis → vertical), on a display: flex container.
A design needs the first item pinned to the far left and the last to the far right, gaps filling the middle. Which one justify-content value?
space-between — it puts zero gap at the outer edges and distributes all the free space between the items.
Spaced practice — do this before Lesson 3. Play a few levels of Flexbox Froggy. It drills exactly justify-content, align-items, and flex-direction by making you write real CSS — retrieval a day later is what turns today's win into something that sticks.

✦ What you earned

justify-content distributes items on the main axis; align-items aligns them on the cross axis (and center + center centers a box); gap spaces them. Given a target layout, you can now name the rule instead of nudging margins.
