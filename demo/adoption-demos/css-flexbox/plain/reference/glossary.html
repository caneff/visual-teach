Glossary

The shared vocabulary of the course. Used the same way in every lesson.

Flex container
An element with display: flex. It lays out its direct children along an axis. The container owns flex-direction, justify-content, align-items, flex-wrap, and gap.
Flex item
A direct child of a flex container. Only immediate children become items; deeper descendants are laid out by their own parents. Items own flex-grow, flex-shrink, flex-basis, align-self, and order.
Main axis
The direction items flow, chosen by flex-direction. With row it runs left→right; with column it runs top→bottom. Not a synonym for “horizontal.” justify-content works this axis.
Cross axis
Always perpendicular to the main axis. With row it runs top→bottom; with column it runs left→right. align-items works this axis.
flex-direction
Container property choosing the main axis: row (default), row-reverse, column, column-reverse.
justify-content
Distributes items along the main axis and shares leftover space: flex-start, flex-end, center, space-between, space-around, space-evenly.
align-items
Positions items on the cross axis: stretch (default), flex-start, flex-end, center, baseline. align-items: center is how you finally center vertically.
gap
Fixed spacing between items, with no margins and no last-child exception. Splits into row-gap and column-gap.
flex-grow
Item property: a unitless ratio for how much leftover main-axis space the item claims. 0 = stay put (default).
flex-shrink
Item property: a unitless ratio for how readily the item shrinks when space is tight. 1 = may shrink (default); 0 = never shrink.
flex-basis
Item property: the starting size on the main axis before growing or shrinking. A length (200px) or auto (use content size).
flex (shorthand)
Writes all three at once: flex: grow shrink basis. Common presets: flex: 1 (grow equally), flex: 0 0 200px (rigid), flex: 0 0 auto (natural size).
flex-wrap
Container property: nowrap (default, one line) vs wrap (items overflow onto new lines) — the basis of a responsive row.
Convention Throughout this course, main axis is written in blue and cross axis in orange so the two never blur together.
