Reference · nomenclature

Glossary

The vocabulary of flexbox. These terms are used consistently across every lesson.

Layout debugging is mostly a vocabulary problem: once you can name the container, the axis, and the basis, you can search for the answer and ask a precise question. These are the words.

Flex container
Any element with display: flex (or inline-flex). It establishes a flex formatting context for its direct children. Introduced in Lesson 1.
Flex item
A direct child of a flex container. Grandchildren are not flex items unless their own parent is also a flex container.
Main axis
The axis flex items lay out along. Its direction is set by flex-direction — horizontal for row, vertical for column. justify-content and the flex sizing properties act on this axis.
Cross axis
The axis perpendicular to the main axis. align-items, align-self, and align-content act here. On a row, this is the vertical direction — which is why cross-axis center is how you center vertically.
Main size / cross size
An item's dimension along the main axis (its width on a row) and along the cross axis (its height on a row). flex-basis sets the starting main size.
Start / end lines
The edges of each axis. flex-start means the start edge of the relevant axis, not "left" — it follows writing direction and flex-direction.
flex-grow
An item's share of leftover main-axis space. 0 = don't grow; higher numbers take proportionally larger shares.
flex-shrink
How readily an item gives space back when items would overflow. 1 = shrink proportionally (default); 0 = refuse to shrink.
flex-basis
An item's main size before grow and shrink are applied. auto uses the content/declared size; a length starts there.
flex (shorthand)
flex: grow shrink basis in one declaration. flex: 1 expands to 1 1 0; flex: auto to 1 1 auto; flex: none to 0 0 auto.
Wrapping
With flex-wrap: wrap, items that don't fit on one line flow onto new lines, turning a single row into a multi-line layout. See Lesson 3.
gap
The minimum gutter between flex items (and between wrapped lines). Replaces the old pattern of margins on children.
Available space
The room left on the main axis after items are laid out at their bases. The justify-content space-* values and flex-grow both distribute this.
Missing a term? Ask me to add it — the glossary is meant to grow with you, and a shared vocabulary makes every future lesson faster.
