Flexbox cheatsheet

Every property from the course, on one page. Built to print.

container properties go on the parent (display: flex). item properties go on a child. The two axes: main = the direction flex-direction points; cross = perpendicular.

container  Set up the box

displayflex — make this a flex container; direct children become items.
flex-directionrow (default) · row-reverse · column · column-reverse. Chooses the main axis.
flex-wrapnowrap (default) · wrap · wrap-reverse. Lets items spill onto new lines.
gapFixed space between items, e.g. 16px or 1rem. Also row-gap / column-gap.

container  Distribute & align

justify-contentmain axis: flex-start · flex-end · center · space-between · space-around · space-evenly.
align-itemscross axis: stretch (default) · flex-start · flex-end · center · baseline.
align-contentSpaces multiple lines on the cross axis (only matters with flex-wrap: wrap).

item  Flexible sizing

flex-growRatio for sharing leftover space. 0 = don't grow (default).
flex-shrinkRatio for shrinking when cramped. 1 = may shrink (default); 0 = never.
flex-basisStarting size on the main axis before grow/shrink, e.g. 200px or auto.
flexShorthand grow shrink basis. Presets: flex: 1 · flex: 0 0 200px · flex: 0 0 auto.
align-selfOverride align-items for one item.
orderReorder a single item visually; default 0.

★  Recipes you'll reuse

Center anythingdisplay:flex; justify-content:center; align-items:center;
Nav: logo ↔ linksdisplay:flex; justify-content:space-between; align-items:center;
Equal columnscontainer display:flex + each child flex:1
Responsive cardsdisplay:flex; flex-wrap:wrap; gap:1rem; + cards flex:1 1 16rem
Vertical stackdisplay:flex; flex-direction:column; gap:.5rem;
The one rule to remember Every flex decision is just: which axis? justify-content and flex-grow work the main axis; align-items works the cross axis. Get the axis right and the property follows.
