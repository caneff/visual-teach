Flexbox cheat sheet

Every property worth memorising, grouped by where it goes and which axis it touches.

Two questions answer almost everything: does this go on the container or the item? and which axis does it touch? The tables below are organised by exactly those two questions.

Set on the container

PropertyAxisValues & effect
display: flex — Makes the element a flex container; its direct children become flex items on a row.
flex-direction sets main row (default) · row-reverse · column · column-reverse. Chooses which way the main axis points; the cross axis is always perpendicular.
justify-content main flex-start · flex-end · center · space-between · space-around · space-evenly. Distributes items along the main axis.
align-items cross stretch (default) · flex-start · flex-end · center · baseline. Aligns items on the cross axis. center = vertical centering on a row.
flex-wrap cross nowrap (default) · wrap · wrap-reverse. Lets items flow onto multiple lines.
gap both e.g. gap: 1rem or gap: 1rem 2rem (row / column). Minimum gutter between items — replaces child margins.
align-content cross Only matters when wrapped onto multiple lines: distributes the lines on the cross axis. Same value vocabulary as justify-content.

Set on an item

PropertyScopeValues & effect
flex shorthand grow shrink basis. Common: flex: 1 (= 1 1 0, equal fill) · flex: 0 0 auto (rigid, natural size) · flex: 1 1 220px (responsive card).
flex-grow main 0 (default, don't grow) · 1+ — share of leftover space to absorb, relative to siblings.
flex-shrink main 1 (default, may shrink) · 0 (never shrink — can overflow). How readily the item gives space back when tight.
flex-basis main auto (use content/width) · a length like 200px. The item's size before grow/shrink.
align-self cross Overrides the container's align-items for this one item. Same values, plus auto.

Recipes you'll reuse

/* Dead-center a box, both axes */
.center { display:flex; justify-content:center; align-items:center; }

/* Navbar: brand left, links right */
.nav { display:flex; justify-content:space-between; align-items:center; gap:1rem; }

/* Equal-width columns that fill the row */
.cols   { display:flex; gap:1rem; }
.cols > * { flex:1; }

/* Responsive card grid: ≥220px each, wrap as needed */
.cards   { display:flex; flex-wrap:wrap; gap:1rem; }
.cards > * { flex:1 1 220px; }

/* Stack on the cross axis: a vertical menu */
.menu { display:flex; flex-direction:column; gap:0.5rem; }
When a property "does nothing" You're almost always talking to the wrong axis. On row: justify-content = horizontal, align-items = vertical. On column they swap. And align-items: center looks inert when the container has no extra cross-axis size to center within.
