Building the major scale, applied

One pattern, started from any note, hands you the seven notes of a key — on purpose.

This is the mission: stop guessing, start constructing. The major scale is a single fixed recipe of whole and half steps. Apply it from C and you get C major. Apply the very same recipe from any other note and you get that key — sharps and flats appear exactly where the pattern demands them.

The recipe

From your starting note — the root — walk this sequence of whole (2 semitones) and half (1 semitone) steps. Seven steps, and the last one returns you to the root an octave up.1

WWH WWW H

W–W–H–W–W–W–H. It splits into two identical halves — W–W–H, a whole step, W–W–H — called tetrachords. Every major key in existence uses this exact shape.

Build 1 — C major (the free one)

Start on C and obey the pattern. Each step lands on a white key, so C major needs no sharps or flats at all — that is what makes it the scale everyone starts with.

Build 2 — G major (where a sharp is born)

Now start on G. First lay out the seven letter names in order, using each letter once: G A B C D E F → G. Then walk the pattern, counting semitones the way you practised in Lesson 2:

GW AW BH CW DW EW F♯H G

Everything matches the natural notes until the sixth step. The pattern says E must rise a whole step (2 semitones) — but E up to natural F is only a half step (1 semitone, the no-black-key pair). One semitone short. So you raise it to F♯, and now E→F♯ is a true whole step. The final F♯→G is the half step that lands home. That single adjustment is the whole reason G major has one sharp.

The only key that changes for G major: F♯. Everything else is a natural — the pattern, not memorisation, told you so.

A flat works the same way. Build from F — F G A B C D E → F — and the pattern forces the fourth note down to B♭, so F major has one flat. Whether a key takes sharps or flats, the recipe is identical; only the root changed. The full set is in the major-scale cheat sheet.

The four-step checklist

  1. Lay out the letters. Seven names from the root, each used once (wrap past G back to A).
  2. Walk W–W–H–W–W–W–H, counting semitones from each note to the next.
  3. Fix each step by adding ♯ or ♭ to a letter when the count is short or long — never rename the letter.
  4. Check the landing: the last H step must return you to the root.
Your win this lesson — and the mission You can now take any note and build its major scale deliberately: lay out the letters, walk W–W–H–W–W–W–H, and add the sharps or flats the pattern demands. No more guessing which notes belong to a key.
