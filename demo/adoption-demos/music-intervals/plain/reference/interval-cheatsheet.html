Interval & major-scale cheatsheet

Everything in this course on one printable page. When you forget, come here — not to a lesson.

1 · The ruler — semitones

One octave = 12 equal semitones, repeating up and down the keyboard. A semitone is one key to the next (white or black). Count gaps from the lower note (0) to name any interval.

3 · The major-scale rule

WWH WWW H

W–W–H–W–W–W–H = semitones 2‑2‑1‑2‑2‑2‑1. Half steps fall between scale steps 3→4 and 7→8. Apply from any root; name each note with the next letter A–G, adding ♯ or ♭ only as the count demands.

Sharps appear going “up” in fifths (G, D, A…); flats going the other way (F, B♭…). Each scale spells one of every letter — never two of the same.

5 · Build-a-scale checklist

  1. Pick the root = scale step 1.
  2. Apply 2‑2‑1‑2‑2‑2‑1 semitones up the keyboard.
  3. Spell with consecutive letters; add ♯/♭ as needed.
  4. Check: did you use each letter A–G exactly once? End on the octave.
One-line summary Twelve semitones repeat; an interval is a count of them; a major scale is the count pattern W‑W‑H‑W‑W‑W‑H from any note.
