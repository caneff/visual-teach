Piano

An interactive, labelled keyboard. Renders a span of semitone keys from a starting note; lessons highlight keys, read clicks, and build exercises on top. Pairs with piano.js.

Labelled white keys (click to name)

data-start="C4" data-keys="25" data-labels="white". Click any key — the readout names it.

One octave, every key labelled

data-keys="13" data-labels="all" data-readout="off".

Highlight states (set by lessons via the API)

setState(i, 'is-root' | 'is-target' | 'is-on') and badge(i, text). Here C is the root and G (7 semitones up) is the target.

Instance API

After init, each block carries el.vtPiano:

  • keys — array of key elements, left→right
  • at(i) / find(abs) — look up a key
  • setState(i, cls) / clear(cls?) — highlight
  • badge(i, text) — drop a small number badge
  • say(html) — write the readout line

Every click also emits a vt-key CustomEvent with { index, pc, name, octave, abs, white, el }.
