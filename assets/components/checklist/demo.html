Component Demo

Checklist

A persisted step-by-step checklist with progress bar. State is saved to localStorage.

Example: Set up a local dev environment

  • Install Node.js Download the LTS release from nodejs.org and run the installer. Verify with node -v.
  • Clone the repository Run git clone https://github.com/example/repo.git and cd into the project directory.
  • Install dependencies Run npm install to fetch all packages listed in package.json.
