# visual-teach

Rich, interactive teaching lessons rendered from a structured block format.

Authoring vocabulary is inspired by the Agent-Native `visual-plan` block system,
but this is a standalone project: no MCP, no hosted service, no agent-feedback
loop. Lessons are local source files rendered by our own thin renderer, with
teaching-first blocks (quizzes, runnable code) as first-class citizens.

See [`docs/visual-plan-analysis.md`](docs/visual-plan-analysis.md) for the
research that motivated the split.
